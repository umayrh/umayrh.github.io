Fibonacci formula

2.1 Given the formula

$F_{n+2} = F_{n+1} + F_{n}$

$u_n = \tau^{n} \therefore \tau^{n+2} = \tau^{n+1} + \tau^{n}\ \Rightarrow \tau^{2} = \tau + 1$
